# Coffee Shop Analytics Dashboard

โปรเจค Dashboard วิเคราะห์ธุรกิจร้านกาแฟ สำหรับผู้บริหาร

## 📁 ไฟล์ในโปรเจค

| ไฟล์ | รายละเอียด |
|-------|-------------|
| `dashboard.html` | Dashboard หลัก (HTML + JS + CSS) |
| `KPI_Coffee_Shop.csv` | รายละเอียด KPI ทั้ง 29 ตัว |
| `coffee_sales_group1_realistic_1500rows.csv` | ข้อมูลยอดขาย 1500 รายการ |
| `design-spec.json` | ข้อกำหนดการออกแบบ |

## 🚀 วิธีใช้งาน

1. เปิดไฟล์ `dashboard.html` ใน Browser
2. Dashboard จะโหลดข้อมูลจาก `coffee_sales_group1_realistic_1500rows.csv` อัตโนมัติ
3. หรือ Import ไฟล์ XLSX/CSV ของคุณเอง

## 📊 Dashboard Features

- **6 มิติ:** ยอดขาย, ผลิตภาพ, ลูกค้า, ผลิตภัณฑ์, ช่องทาง, เวลา
- **29 KPIs** พร้อมคำอธิบายสำหรับ CEO
- **Cross Filtering** - คลิกเพื่อกรองข้อมูล
- **KPI Info Popup** - คลิก ℹ️ เพื่อดูรายละเอียด KPI
- **Bento Box Layout** - ขนาด 1550px
- **Responsive Design** - รองรับ Import XLSX/CSV

## 🛠️ เทคโนโลยี

- HTML5 + CSS3 + JavaScript
- Chart.js (Charts)
- XLSX.js (Import Excel)
- Google Fonts (Prompt)
